Deployed cloud-native architecture on AWS EKS with Kubernetes orchestration featuring DaemonSet pattern for distributed LLM access across 3 nodes, implementing RESTful API with http4s and Cats Effect serving 5 microservices (Query, Evidence, Explore, Jobs, Explain) with sub-second latency and OpenAPI documentation.
